<p> Este é um projeto simples de um relógio digital desenvolvido com HTML, CSS e JavaScript. Ele exibe a hora atual em tempo real no formato de horas, minutos e segundos, utilizando o horário do sistema do usuário.
O projeto é composto por três arquivos principais: um arquivo HTML responsável pela estrutura da página, um arquivo CSS para a estilização e um arquivo JavaScript que contém a lógica de funcionamento do relógio. 
A atualização da hora ocorre automaticamente a cada segundo.
</p>
